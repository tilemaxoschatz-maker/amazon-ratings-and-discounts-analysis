

# Amazon Ratings and Discounts Analysis

## Project Objective

Analyze Amazon product data and investigate the relationship between ratings, discounts and customer engagement.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- Most products have ratings between 4.0 and 4.5.
- Discount percentage shows weak correlation with ratings.
- Certain categories consistently achieve higher ratings.
- Product ratings remain relatively stable regardless of discount level.

## Recommendations

- Focus on product quality rather than aggressive discounting.
- Investigate highly rated categories.
- Further analyze products with low ratings despite large discounts.
